HTML

HTML (Hypertext Markup Language) is a language used for creating and structuring web pages and applications. It uses a system of tags to format and organize content, such as text, images, and other media, on a web page. HTML allows for the creation of interactive and dynamic web pages, and is the cornerstone of the modern web.

More Info

CSS

CSS (Cascading Style Sheets) is a language used for styling and formatting the content on web pages. It allows for the separation of content (written in HTML) from presentation, making it easier to maintain and update the visual style of a website. CSS uses a set of rules to determine how the content on a web page should be displayed, including font, color, and layout. It enables developers to create visually-appealing and consistent websites that can be easily accessed on a variety of devices.

More Info

BOOTSTRAP

Bootstrap is a popular front-end framework used for developing responsive, mobile-first websites and applications. It includes a set of predefined CSS and JavaScript styles and components that make it easier to quickly create consistent, user-friendly interfaces. Bootstrap includes a grid system that allows for easy layout of content on a web page, as well as pre-designed components such as buttons, forms, and navigation bars. It is a widely-used tool among web developers, and is known for its ease of use and flexibility.
